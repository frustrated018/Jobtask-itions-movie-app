## Run Locally

Clone the project

```bash
  git clone https://github.com/frustrated018/Jobtask-itions-movie-app.git
```

Go to the project directory

```bash
  cd Jobtask-itions-movie-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
